# 시스템 요구사항

## 현재 구성: Sm_AICoder (CPU 전용)

| 항목 | 요구사항 |
| ---- | -------- |
| OS | Windows 10/11 |
| Python | 3.10 ~ 3.13 |
| RAM | 8GB 이상 (16GB 권장) |
| 디스크 | 6GB 이상 여유 (모델 ~4.4GB) |
| GPU | 불필요 (CPU 전용 실행) |

### 사용 모델

| 모델 | 크기 | 용도 |
| ---- | ---- | ---- |
| Qwen2.5-Coder-7B-Instruct Q4_K_M | ~4.4GB | 자연어 → 코드 (기본) |
| Qwen2.5-Coder-1.5B-Instruct Q4_K_M | ~1.0GB | 경량 테스트용 |
| DeepSeek-Coder-6.7B-Instruct Q4_K_M | ~3.8GB | 대안 |

> 모든 모델은 GGUF 양자화 포맷이며, llama-cpp-python으로 CPU에서 실행됩니다.

---

## CPU 성능 참고

| 항목 | 기대치 |
| ---- | ------ |
| 모델 로딩 | 30~60초 |
| 토큰 생성 속도 | ~1-5 토큰/초 |
| 응답 시간 (짧은 코드) | 30~60초 |
| 응답 시간 (긴 코드) | 1~3분 |

> NVMe SSD 사용 시 모델 로딩이 빠릅니다.

---

## (참고) GPU 사용 시 VRAM 요구사항

GPU가 있는 경우 `N_GPU_LAYERS`를 설정하면 추론 속도를 높일 수 있습니다.

| 모델 | Q4_K_M VRAM | 비양자화 VRAM |
| ---- | ----------- | ------------- |
| Qwen2.5-Coder-1.5B | ~1.5GB | ~3GB |
| Qwen2.5-Coder-7B | ~4.5GB | ~14GB |
