# 시스템 요구사항

## GPU VRAM 요구사항

### StarCoder2-3B (`bigcode/starcoder2-3b`)

| 정밀도 | VRAM |
|--------|------|
| float32 (full) | ~12.6 GB |
| bfloat16 | ~6.3 GB |
| 8-bit 양자화 | ~3.4 GB |
| 4-bit 양자화 | ~2.0 GB |

### StarCoder2-7B (`bigcode/starcoder2-7b`)

| 정밀도 | VRAM |
|--------|------|
| float32 (full) | ~29.2 GB |
| bfloat16 | ~14.6 GB |
| 8-bit 양자화 | ~7.7 GB |
| 4-bit 양자화 | ~4.2 GB |

### StarCoder2-15B (`bigcode/starcoder2-15b`)

| 정밀도 | VRAM |
|--------|------|
| bfloat16 | ~32 GB |
| 8-bit 양자화 | ~17 GB |
| 4-bit 양자화 | ~9 GB |

## 시스템 RAM 요구사항

| 모델 | 최소 RAM | 권장 RAM |
|------|----------|----------|
| 3B | 8 GB | 16 GB |
| 7B | 16 GB | 32 GB |
| 15B | 32 GB | 64 GB |

## 디스크 공간

| 모델 | 디스크 공간 |
|------|------------|
| 3B | ~10 GB |
| 7B | ~15 GB |
| 15B | ~50 GB |

> **권장**: NVMe SSD (모델 로딩 속도 향상)

## GPU 권장 사양

| 상황 | 권장 GPU |
|------|---------|
| 입문 / 테스트 (3B 4-bit) | RTX 3060 12GB 이상 |
| 실용 (7B bfloat16) | RTX 3080 / 4080 |
| 고성능 (15B 4-bit) | RTX 3090 / 4090 |
| 최상 (15B full) | NVIDIA A100 40GB |

## CPU 전용 실행 (GPU 없음)

GPU가 없어도 실행 가능하지만 매우 느림 (토큰 생성 속도 ~1-5 토큰/초):

- 3B 모델: RAM 12GB 이상 필요
- 7B 모델: RAM 28GB 이상 필요

> **CPU 전용 대안**: llama.cpp + GGUF 양자화 모델 사용 권장
