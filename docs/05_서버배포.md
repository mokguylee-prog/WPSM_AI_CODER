# 서버 배포 가이드

로컬 API 서버로 StarCoder2를 실행하면 VS Code 확장, curl, Python 등에서 호출 가능합니다.

---

## 옵션 비교

| 방법 | 속도 | 설치 난이도 | Windows 지원 | GPU 필요 |
|------|------|------------|-------------|---------|
| vLLM | 매우 빠름 | 중간 | WSL2 필요 | 필수 |
| llama.cpp | 빠름 | 쉬움 | 직접 지원 | 선택 |
| TGI (Docker) | 빠름 | Docker 필요 | Docker | 필수 |

---

## 옵션 A: llama.cpp (Windows 직접 지원, 권장)

### 설치

```bash
# Windows용 사전 빌드 바이너리 다운로드
# https://github.com/ggml-org/llama.cpp/releases
# llama-b<번호>-bin-win-cuda-cu12.1-x64.zip 다운로드

# 또는 Python 바인딩 설치
pip install llama-cpp-python --extra-index-url https://abetlen.github.io/llama-cpp-python/whl/cu121
```

### GGUF 모델 다운로드

```bash
# 3B 4-bit 양자화 (~2GB)
huggingface-cli download second-state/StarCoder2-3B-GGUF \
    --include "starcoder2-3b-Q4_K_M.gguf" \
    --local-dir ./models/gguf

# 7B 4-bit 양자화 (~4GB)
huggingface-cli download second-state/StarCoder2-7B-GGUF \
    --include "starcoder2-7b-Q4_K_M.gguf" \
    --local-dir ./models/gguf
```

### 서버 실행

```bash
# GPU 레이어 33개 오프로드, 컨텍스트 8192
./llama-server.exe \
    -m ./models/gguf/starcoder2-7b-Q4_K_M.gguf \
    -ngl 33 \
    -c 8192 \
    --port 8080
```

### Python에서 호출

```python
import requests

prompt = """#include <stdio.h>

int bubble_sort(int arr[], int n) {
    for (int i = 0; i < n - 1; i++) {"""

response = requests.post(
    "http://localhost:8080/completion",
    json={
        "prompt": prompt,
        "n_predict": 256,
        "temperature": 0.2,
        "top_p": 0.95,
        "stop": ["\n}\n\n"]
    }
)

print(response.json()["content"])
```

## 옵션 B: vLLM (WSL2/Linux, 고성능)

### 설치 (WSL2 Ubuntu)

```bash
pip install vllm
```

### 서버 실행

```bash
# OpenAI 호환 API 서버
vllm serve bigcode/starcoder2-7b \
    --gpu-memory-utilization 0.85 \
    --max-model-len 8192 \
    --port 8000
```

### Python에서 호출 (OpenAI 호환)

```python
from openai import OpenAI

client = OpenAI(
    base_url="http://localhost:8000/v1",
    api_key="dummy"
)

prompt = """int binary_search(int arr[], int n, int target) {
    int left = 0, right = n - 1;"""

completion = client.completions.create(
    model="bigcode/starcoder2-7b",
    prompt=prompt,
    max_tokens=256,
    temperature=0.2
)

print(completion.choices[0].text)
```

## 옵션 C: Python 내장 서버 (FastAPI)

```bash
python scripts/api_server.py
```

서버가 `http://localhost:8888` 에서 실행됩니다.

```bash
# curl 테스트
curl -X POST http://localhost:8888/generate \
    -H "Content-Type: application/json" \
    -d '{"prompt": "int factorial(int n) {\n    if (n <= 1)\n        return 1;\n", "max_tokens": 128}'
```

## VS Code 연동

[Continue.dev](https://www.continue.dev/) 확장 설치 후 설정:

```json
// ~/.continue/config.json
{
  "models": [
    {
      "title": "StarCoder2-7B Local",
      "provider": "openai",
      "model": "bigcode/starcoder2-7b",
      "apiBase": "http://localhost:8000/v1",
      "apiKey": "dummy"
    }
  ],
  "tabAutocompleteModel": {
    "title": "StarCoder2 Autocomplete",
    "provider": "openai",
    "model": "bigcode/starcoder2-7b",
    "apiBase": "http://localhost:8000/v1"
  }
}
```
