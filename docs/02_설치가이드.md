# 설치 가이드

> **검증된 환경**: Windows 11, Python 3.13, CPU 전용

## 1단계: Python 가상환경 생성

Python 3.10 ~ 3.13 필요 (3.14는 llama-cpp-python 호환 미확인)

```powershell
# 가상환경 생성
python -m venv venv

# 활성화 (Windows PowerShell)
venv\Scripts\activate

# pip 업그레이드
python -m pip install --upgrade pip
```

---

## 2단계: 패키지 설치

```powershell
pip install -r requirements.txt
```

requirements.txt 내용:

```text
llama-cpp-python>=0.2.90
fastapi>=0.110.0
uvicorn>=0.27.0
pydantic>=2.0.0
requests>=2.31.0
huggingface_hub>=0.21.0
```

> **llama-cpp-python 빌드 실패 시** (MSVC 인코딩 에러):
> `pip install llama-cpp-python==0.3.8 --prefer-binary`

---

## 3단계: 모델 다운로드

```powershell
venv\Scripts\python.exe server\scripts\download_model.py
```

- 기본 모델: Qwen2.5-Coder-7B-Instruct Q4_K_M (~4.4GB)
- 저장 위치: `Sm_AICoder/models/gguf/`
- HuggingFace 로그인 불필요 (공개 모델)

다른 모델을 선택하려면:

```powershell
# 경량 모델 (1.0GB)
venv\Scripts\python.exe server\scripts\download_model.py qwen2.5-coder-1.5b

# DeepSeek 대안 (3.8GB)
venv\Scripts\python.exe server\scripts\download_model.py deepseek-coder-6.7b
```

---

## 4단계: 설치 검증

```powershell
venv\Scripts\python.exe server\scripts\check_env.py
```

모든 항목이 OK로 나오면 설치 완료.

---

## 5단계: 서버 시작

```powershell
.\\start_server.ps1
```

웹 대시보드: `http://localhost:8888`
