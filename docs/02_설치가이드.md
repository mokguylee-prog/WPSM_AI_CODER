# 설치 가이드

> **실제 검증된 환경**: Windows 11, Python 3.12, CPU 전용, D 드라이브

## 1단계: Python 환경 설정

**Python 3.12 필수** (3.13은 llama-cpp-python 사전 빌드 wheel 미지원)

```bash
# Python 버전 확인
py --list

# 3.12로 D 드라이브에 가상환경 생성
py -3.12 -m venv D:\StarCoder2\venv

# 활성화 (Windows)
venv\Scripts\activate

# 활성화 (Linux/macOS/WSL2)
source venv/bin/activate
```

---

## 2단계: CUDA 설치 (NVIDIA GPU 사용 시)

### Windows CUDA 설치

1. **NVIDIA 드라이버 최신 버전 설치**
   - https://www.nvidia.com/Download/index.aspx

2. **CUDA Toolkit 설치** (12.1 권장)
   - https://developer.nvidia.com/cuda-downloads
   - OS: Windows → Architecture: x86_64 → Version: 11 or 12

3. **설치 확인**
```bash
nvidia-smi        # GPU 상태 확인
nvcc --version    # CUDA 버전 확인
```

## 3단계: PyTorch 설치

```bash
# CUDA 12.1 버전
pip install torch torchvision torchaudio --index-url https://download.pytorch.org/whl/cu121

# CUDA 11.8 버전
pip install torch torchvision torchaudio --index-url https://download.pytorch.org/whl/cu118

# CPU 전용
pip install torch torchvision torchaudio
```

```python
# PyTorch + CUDA 설치 확인
import torch
print(f"PyTorch 버전: {torch.__version__}")
print(f"CUDA 사용 가능: {torch.cuda.is_available()}")
print(f"GPU 이름: {torch.cuda.get_device_name(0) if torch.cuda.is_available() else 'N/A'}")
print(f"GPU VRAM: {torch.cuda.get_device_properties(0).total_memory // 1024**3} GB")
```

## 4단계: HuggingFace 패키지 설치

```bash
pip install transformers>=4.39.0
pip install accelerate>=0.27.0
pip install huggingface_hub
```

## 5단계: 양자화 패키지 설치 (선택, VRAM 절약)

```bash
# bitsandbytes (4-bit/8-bit 양자화)
pip install bitsandbytes>=0.43.0

# Windows에서 bitsandbytes 설치 문제 발생 시:
pip install bitsandbytes --prefer-binary
```

> **주의**: bitsandbytes는 Windows에서 설치가 까다로울 수 있음.
> 문제 발생 시 WSL2 환경 사용 권장.

## 7단계: HuggingFace 로그인

StarCoder2 모델은 무료지만 사용 동의가 필요합니다.

1. https://huggingface.co 계정 생성
2. https://huggingface.co/bigcode/starcoder2-7b 에서 "Agree and access repository" 클릭
3. https://huggingface.co/settings/tokens 에서 Access Token 생성 (Read 권한)

```bash
huggingface-cli login
# 토큰 입력 프롬프트가 나타남
```

또는 Python에서:

```python
from huggingface_hub import login
login(token="hf_xxxxxxxxxxxxxxxxxxxx")
```
