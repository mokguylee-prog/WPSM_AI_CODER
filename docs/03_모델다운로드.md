# 모델 다운로드

## 자동 다운로드 (권장)

```powershell
venv\Scripts\python.exe server\scripts\download_model.py
```

기본으로 Qwen2.5-Coder-7B-Instruct Q4_K_M (~4.4GB)을 다운로드합니다.
저장 위치: `Sm_AICoder/models/gguf/`

---

## 다른 모델 선택

```powershell
# 사용 가능한 모델 목록 확인 후 선택
venv\Scripts\python.exe server\scripts\download_model.py qwen2.5-coder-1.5b
venv\Scripts\python.exe server\scripts\download_model.py deepseek-coder-6.7b
```

| 모델 키 | 모델명 | 크기 | 설명 |
| ------- | ------ | ---- | ---- |
| `qwen2.5-coder-7b` | Qwen2.5-Coder-7B-Instruct Q4_K_M | ~4.4GB | 권장, 품질 좋음 |
| `qwen2.5-coder-1.5b` | Qwen2.5-Coder-1.5B-Instruct Q4_K_M | ~1.0GB | 경량, 빠름 |
| `deepseek-coder-6.7b` | DeepSeek-Coder-6.7B-Instruct Q4_K_M | ~3.8GB | 대안 |

> 모든 모델은 HuggingFace Hub에서 다운로드되며 로그인 불필요 (공개 모델)

---

## 수동 다운로드 (huggingface-cli)

```bash
pip install huggingface_hub[cli]

# Qwen2.5-Coder-7B GGUF
huggingface-cli download Qwen/Qwen2.5-Coder-7B-Instruct-GGUF \
    --include "qwen2.5-coder-7b-instruct-q4_k_m.gguf" \
    --local-dir ./Sm_AICoder/models/gguf
```

---

## 모델 자동 선택

서버는 시작 시 `Sm_AICoder/models/gguf/` 폴더에서 **가장 큰 `.gguf` 파일**을 자동으로 선택합니다.
여러 모델을 다운로드해 두고, 원하는 모델만 남겨두면 됩니다.

---

## 다운로드 속도 참고

| 모델 | 파일 크기 | 100Mbps | 1Gbps |
| ---- | --------- | ------- | ----- |
| 1.5B Q4_K_M | ~1.0GB | ~2분 | ~10초 |
| 7B Q4_K_M | ~4.4GB | ~6분 | ~40초 |
| 6.7B Q4_K_M | ~3.8GB | ~5분 | ~35초 |
