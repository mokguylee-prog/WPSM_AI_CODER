## 🤖 Claude

## 학습 필요 없습니다

StarCoder2는 **이미 학습이 완료된 모델**입니다.

HuggingFace에서 다운받으면 바로 C 코드 생성이 가능합니다.

---

### 학습 현황

| 항목 | 내용 |
|------|------|
| 학습 데이터 | The Stack v2 (600+ 언어, C 포함) |
| 학습 토큰 수 | 3B~4.3T 토큰 (엄청난 양) |
| C 코드 포함 여부 | 포함됨 (GitHub의 수백만 C 파일) |
| 학습 주체 | BigCode (HuggingFace + ServiceNow) |
| 학습 완료 시점 | 2024년 초 (이미 완료) |

---

### 그러면 뭘 하면 되나?

```
다운로드 → 로드 → 프롬프트 입력 → 코드 완성 출력
```

끝입니다. GPU에 올리고 쓰면 됩니다.

---

### 선택적으로 추가 학습(Fine-tuning)이 필요한 경우

아래 상황이면 LoRA로 추가 학습을 고려할 수 있지만, **지금 당장은 불필요**합니다:

- 회사 내부 코딩 스타일/규칙을 강하게 따르게 하고 싶을 때
- 특정 라이브러리나 도메인(임베디드, RTOS 등)에 특화시키고 싶을 때
- 일반 C보다 훨씬 더 높은 정확도가 필요할 때

**지금은 그냥 다운받아서 쓰면 됩니다.**

## 방법 1: llama.cpp + GGUF (가장 권장)

CPU에서도 쓸 만한 속도로 실행됩니다.

```bash
# 1. llama.cpp Python 바인딩 설치
pip install llama-cpp-python

# 2. GGUF 모델 다운로드 (3B, Q4 양자화 → 약 2GB)
huggingface-cli download second-state/StarCoder2-3B-GGUF \
    --include "starcoder2-3b-Q4_K_M.gguf" \
    --local-dir ./models/gguf

# 3. 실행
python scripts/generate_cpu_gguf.py
```

**CPU 속도 예상**: 토큰 5~15개/초 (Core i7 기준)
→ 함수 하나 완성에 약 10~30초

---

## 방법 2: 클라우드 (무료 GPU 사용)

### 우리가 StarCoder2에 준 것 (프롬프트 - 미완성 코드)

```c
// Recursive factorial
int factorial(int n) {
    if (n <= 1)
        return 1;
    ← 여기서 끊음
```

절반만 작성한 코드를 입력으로 줬습니다.

---

### StarCoder2가 나머지를 채워서 돌려준 것

```c
// Recursive factorial
int factorial(int n) {
    if (n <= 1)
        return 1;
    else
        return n * factorial(n - 1);   ← AI가 추가
}                                       ← AI가 추가

// Iterative factorial                  ← AI가 알아서 추가
int factorial_iter(int n) {
    int result = 1;
    while (n > 1) {
        result *= n;
        n--;
    }
    return result;
}
```

---

### 팩토리얼이란?

`5!` = 5 × 4 × 3 × 2 × 1 = **120**

| 방식 | 설명 |
|------|------|
| **재귀형** (recursive) | 함수가 자기 자신을 다시 호출 |
| **반복형** (iterative) | for/while 루프로 계산 |

---

### 요약

우리는 함수를 **절반만** 입력했는데, StarCoder2가 **나머지 구현을 자동으로 완성**했고, 심지어 요청하지도 않은 **반복형 버전까지 추가로 생성**했다는 뜻입니다.

즉, 기본 동작은 성공한 겁니다.
